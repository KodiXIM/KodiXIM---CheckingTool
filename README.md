🧊 Full Description (with apps listed)

KodiXIM Tool Hub is a one-click utility that downloads and organises a set of trusted system and forensic tools into a dedicated workspace. It automatically installs and extracts everything so the tools are ready to use instantly.

The hub includes:

JumpListExplorer – view and analyse Windows jump lists
Disk Investigator – examine raw disk data and recover information
Everything – ultra-fast file search engine for Windows
BAM Parser – analyse Background Activity Moderator data
System Informer – advanced system and process monitoring
WinPrefetchView – inspect Windows prefetch files
WinDefLogView – view Windows Defender logs
LastActivityView – track recent system activity

All tools are downloaded from their official sources and stored inside the KodiXIM folder, with archives automatically extracted for convenience.
